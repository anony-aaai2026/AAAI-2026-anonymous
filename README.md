# AAAI-2026-anonymous

This repository contains code to reproduce the results in our AAAI 2026 submission.

## Requirements

- Python 3.11.7
- Required packages (install with pip):

``` bash
pip install -r requirements.txt
```
## Running the code
- Toy example:
```
python toy_example.py
```

- Real world data:
```
python real_world_data.py
```
